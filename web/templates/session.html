Import cookies

Export your x.com cookies with a browser extension while logged in, then upload the file. Works with Netscape cookies.txt ("Get cookies.txt LOCALLY") or JSON ("Cookie-Editor"). The export is plaintext, so no keyring is involved — this is the container-friendly alternative to import-profile.

On a host with Chrome, you can instead run python main.py import-profile (decrypts your Chrome session via the keyring). That path can't run inside a container — use the upload above.
